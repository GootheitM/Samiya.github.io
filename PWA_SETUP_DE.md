# LernQuest als App nutzen (iPhone, Android, PC)

## Was ist bereits eingebaut?
- `Lernwebsite.html` wurde als PWA vorbereitet.
- `manifest.webmanifest` ist vorhanden.
- `sw.js` (Service Worker) ist vorhanden.
- App-Icons: `icon-192.svg` und `icon-512.svg`.

## Wichtig
Damit Installation und Offline-Funktion sauber laufen, muss die Seite über **HTTPS** geöffnet werden (nicht direkt als lokale Datei).

## Kostenlos online stellen (empfohlen: GitHub Pages)
1. GitHub-Repository erstellen.
2. Alle Dateien aus diesem Ordner hochladen.
3. In GitHub: `Settings` -> `Pages` -> Branch auswählen -> speichern.
4. Nach 1-2 Minuten erhältst du eine öffentliche URL.

## Installation auf iPhone (Safari)
1. Öffne die GitHub-Pages-URL in Safari.
2. Teile-Menü öffnen.
3. `Zum Home-Bildschirm` wählen.
4. Fertig: LernQuest erscheint wie eine App.

## Installation auf Android (Chrome)
1. Öffne die URL in Chrome.
2. `App installieren` oder `Zum Startbildschirm hinzufügen` wählen.
3. Fertig: LernQuest startet im App-Modus.

## Installation auf Laptop/PC (Chrome/Edge)
1. Öffne die URL im Browser.
2. In der Adressleiste auf das Installieren-Symbol klicken (oder Browser-Menü -> App installieren).
3. Fertig: startet wie eine normale Desktop-App.
